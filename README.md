🚀 LaunchLens

LaunchLens is a modern platform that tracks and showcases newly launched products from Product Hunt in real time. It helps makers, developers, startup founders, and tech enthusiasts discover trending products through a powerful analytics dashboard.

![Dashboard Preview](./assets/dashboard-preview.png)

## ✨ Features

### 📅 Launch Tracking

* Daily launched products
* Weekly trending products
* Monthly top-performing launches
* Product of the Day highlights

### 🔍 Smart Search

* Search products instantly
* Search by category
* Search by keywords and tags

### 🎯 Advanced Filters

* Daily launches
* Weekly launches
* Monthly launches
* Free products
* Paid products
* Freemium products
* Category filters

### 📊 Analytics Dashboard

* Total launches
* Total upvotes
* Product growth tracking
* Launch trends visualization
* Category distribution charts

### 🏆 Product Rankings

* Top products by upvotes
* Trending products
* Most discussed products
* Product Hunt leaders

### 🎨 Modern UI

* 3D dashboard design
* Dark mode interface
* Glassmorphism effects
* Responsive design
* Mobile-friendly experience

## ⚡ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/launchlens.git
```

### Navigate

```bash
cd launchlens
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file:

```env
PRODUCT_HUNT_ACCESS_TOKEN=your_token
DATABASE_URL=your_database_url
NEXT_PUBLIC_APP_URL=http://localhost:3000
```
